# Installateurs

Ce répertoire contient les scripts d'installation pour le TP World3.

## Linux
- Fichier : `setuponlinux.py`
- Compatible avec **Ubuntu 20.04 à 24.04**.
- Utilisation :
  - **Version française :**  
    `python3 setuponlinux.py`
  - **Version anglaise :**  
    `python3 setuponlinux.py en`

## Windows
- Fichier : `setuponwindows.py`
- Testé sous **Windows 10** et **Windows 11**.
- Utilisation :
  - **Version française :**  
    `python setuponwindows.py`
  - **Version anglaise :**  
    `python setuponwindows.py en`

Ces installateurs créent automatiquement un environnement Python, téléchargent le notebook correspondant et l'ouvrent dans Jupyter. Ils ne suppriment ni n'écrasent les fichiers existants de l'utilisateur.
