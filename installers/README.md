# Installers

This directory contains the setup scripts for the TP World3 practical.

## Linux
- File: `setuponlinux.py`
- Supports **Ubuntu 20.04 to 24.04**.
- Usage:
  - **French version:**  
    `python3 setuponlinux.py`
  - **English version:**  
    `python3 setuponlinux.py en`

## Windows
- File: `setuponwindows.py`
- Tested on **Windows 10** and **Windows 11**.
- Usage:
  - **French version:**  
    `python setuponwindows.py`
  - **English version:**  
    `python setuponwindows.py en`

These installers automatically create a Python environment, download the correct notebook, and open it in Jupyter. They do not delete existing files or overwrite user work.
